# Firewall Simulator

## Overview
This simulator creates an animated wall that allows protocols based on the rules mentioned. It features a queue of protocols, a speed rate, and a real-time dashboard.

## Features
- Animated wall simulation
- Protocol management based on defined rules
- Real-time dashboard for monitoring
- Adjustable speed for protocol processing

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/firewall-simulator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd firewall-simulator
   ```
3. Open `index.html` in your web browser to view the simulator.

## Usage
- Follow the on-screen instructions to add protocols to the queue.
- Adjust the speed settings to see how the wall reacts to different protocols.
- Monitor the real-time dashboard for updates on protocol processing.
